# PR#1-3: コード変換ユーティリティ実装

## 概要
JV-Linkのコード値を人間が読める文字列に変換する機能を実装しました。競馬場コード、性別コード、トラックコード、グレードコード、馬場状態コード、天候コード、重量種別コードの7種類のコード変換関数を提供します。

## 変更内容

### 新規作成ファイル
- `jvlink_db_python/utils/code_converter.py`: コード変換関数の実装
  - `jyo_code_to_name()`: 競馬場コード→競馬場名（中央10、地方32、海外76の計118競馬場に対応）
  - `sex_code_to_str()`: 性別コード→性別文字列（牡・牝・セ）
  - `track_code_to_str()`: トラックコード→トラック種別文字列（芝・ダート・障害の29パターン）
  - `grade_code_to_str()`: グレードコード→グレード文字列（G1～G3、J･G1～J･G3、リステッド等）
  - `baba_code_to_str()`: 馬場状態コード→馬場状態文字列（良・稍・重・不）
  - `weather_code_to_str()`: 天候コード→天候文字列（晴・曇・雨・小雨・雪・小雪）
  - `weight_type_code_to_str()`: 重量種別コード→重量種別文字列（ハンデ・別定・馬齢・定量）

### 既存ファイルの変更
- `jvlink_db_python/utils/__init__.py`: コード変換関数のエクスポート追加
- `doc/PLAN.md`: PR#1-3の作業内容を更新
